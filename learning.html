Which versio? Which distribution? How do I install...?

Pick a version of python and start learning. It's easy to change your mind later. They aren't radically different. You can even try out different versions of python online before deciding what to install.

Online Python Interpretors

  • PythonAnywhere: many different python shells. python, ipython, pypy (requires a free account)
  • Wakari: scientific python stack with ipython notebook support. (requires a free account)

Learning Python

  • The official Python tutorial: This tutorial is very comprehensive, but it sometimes throws in unnecessarily advanced concepts. This is a solid tutorial for people with prior programming experience, and is still a good tutorial for new programmers if you are willing to skip over the advanced concepts and come back to them later.
  • Learn Python the Hard Way: Despite the name, some people find this to be a gentler introduction than the official Python tutorial.
  • How to Think Like a Computer Scientist: Learning with Python: This interactive tutorial does a particularly good job of introducing programming concepts and terminology.
  • Learn programming by visualizing code execution: Online Python Tutor
  • Software Carpentry: Programming With Python: "The best way to learn how to program is to do something useful, so this introduction to Python is built around a common scientific task: data analysis."
